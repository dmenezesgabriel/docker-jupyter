# Jupyter Analytics

## Jypyter lab

### Features

- Oh my ZSH terminal
- Dracula theme
- Markdown cells spellcheck (en_US)
- Code completion
- Code formatter
- Table fo contents
- Collapsible headers

### Requirements

- Docker
- Docker Compose

### Usage

1. Build the image:
   ```sh
   make build-lab
   ```
2. Run the container:
   ```sh
   make run-lab
   ```

- **Activate zsh**:
  Use the command below on terminal

```sh
zsh
```
